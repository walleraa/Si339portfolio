Procrastination Portfolio

What is this?

It's a portfolio of all the ways I personally waste my time. I could make a form here that uses a backend to ask you how you waste your time and allow you to put an image here yourself, but I didn't feel like going through making routes and setting up an API so instead I'm wasting more of my own time trying to make an Etch A Sketch in JavaScript. I'm going to regret this.

If you're viewing this on mobile, the poorly made Etch-A-Sketch will not be available, as the screen size is too small. Sorry for the inconvenience!

Since I wasn't a big fan of the wheels on the original Etch-A-Sketch, my version just has directional buttons.

If you want to use keys, you can! It will read your keystrokes (You can use WASD or the arrow keys).

To make things easier, there is also a reset button to move your position back to the top left.

You can use the C and R keys to clear and reset, respectively

Below are some buttons to change the color of the background and line, try it out and have fun!

If your drawing seems to get erased after changing colors, keep drawing, it'll reappear!
